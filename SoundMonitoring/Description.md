# SoundMonitoring
Able to be ran standalone on a Raspberry Pi 3B+ (with pycaw dropped entirely in ReadSound.py), monitoring my surrounding sound levels throughout the day.

Runs a lightweight Flask web server that displays the sensor readings in a dashboard. It has a live-updating graph, statistics (peak, average, minimum), and a refrence table to interpret dB levels.
